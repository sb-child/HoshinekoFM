# Materials File Explorer

A modern, "Performance-First" file manager built with Electron, React, and Material Design 3.

## Features

- **Material Design 3 Interface**: sleek, modern UI with dynamic theming.
- **Performance First**: Background size calculations, virtualized lists for large directories.
- **Dual Pane / Tabs**: (Coming soon/Partially implemented) Tabbed navigation.
- **Omnibar**: Unified search and address bar compatible with `fd` and standard shell commands.
- **Terminal Integration**: deeply integrated terminal support.
- **Preview Support**: Quick look for common file types.

### Manual Build

1. Clone the repository:
   ```bash
   git clone https://github.com/bhimio1/material-3-file-explorer.git
   cd material-3-file-explorer
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run in development mode:
   ```bash
   npm run dev
   npm run electron:dev
   ```

4. Build for production:
   ```bash
   npm run electron:build
   ```
